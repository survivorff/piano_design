# 练习日志模板（Practice Journal Templates）

> "没有记录的练习，进步是模糊的。" 写练习日志能让你**目标清晰、追踪进度、发现规律、保持动力**。本篇提供可直接复制使用的模板。复制到你的笔记软件，每天/每周填写。

---

## 一、为什么要写练习日志？

1. **目标导向**：逼自己练前想清楚"今天练什么、要达到什么"。
2. **追踪进度**：清楚看到自己一周/一月的成长，对抗"感觉没进步"的沮丧。
3. **发现问题规律**：哪些难点反复出现？哪些时间段效率高？
4. **保持动力**：打卡和记录成果本身就有激励作用。
5. **高效复盘**：下次练习能精准接续上次的进度，不浪费时间重新摸索。

---

## 二、每日练习日志模板

```markdown
## 📅 日期：____年__月__日   ⏱️ 时长：___分钟

### 🎯 今日目标（练前填，要具体！）
- 目标1：____（如：巴赫创意曲16-19小节左手跳音，慢练到无错且放松）
- 目标2：____

### 📝 今日内容
| 内容 | 时长 | 速度/进度 | 备注 |
|------|------|-----------|------|
| 热身（音阶/放松） |  |  |  |
| 技术（练习曲） |  |  |  |
| 曲目攻坚 |  |  |  |
| 整合/音乐性 |  |  |  |
| 视奏/自由弹 |  |  |  |

### ✅ 今日收获
- ____（如：第16小节的跳音终于稳了；C大调音阶提到了80BPM）

### ⚠️ 遇到的问题 / 明日重点
- ____（如：合手时左右手对不齐第18小节，明天单独抠）

### 😊 状态自评（1-5）
- 专注度：___  放松度：___  愉悦度：___
```

## 三、极简版（懒人每日打卡）

适合时间紧、不想写太多的人：

```markdown
[日期] 练了__分钟 | 练了：____ | 进步：____ | 待解决：____
```

例：
```
6/8 练了30分 | 巴赫创意曲+C大调音阶 | 创意曲右手顺了 | 左手第18小节还卡
```

## 七、使用建议

1. **选适合你的颗粒度**：忙就用极简版，认真时用完整版。**坚持记 > 记得详细**。
2. **练前填目标，练后填收获**——这是日志最大的价值所在。
3. **每周花 5 分钟复盘**，每月回看，你会清楚看到成长轨迹。
4. **诚实**：记录真实情况（包括偷懒和卡壳），日志是给自己的，不是表演。
5. 可以配合手机日历/笔记 App/表格软件，设每日提醒。

> 🔑 配合 [科学练琴原则](./01-科学练琴原则.md)（明确目标、攻弱点）和 [分阶段练习计划](./02-分阶段练习计划.md) 使用，日志就是把这些原则落地的工具。

---

## 八、小结

- 练习日志让你**目标清晰、追踪进度、发现规律、保持动力、高效复盘**。
- 提供五种模板：**每日（完整/极简）、每周复盘、曲目攻坚追踪、月度/长期目标**。
- 核心用法：**练前填目标、练后填收获、每周复盘**。
- **坚持记录比记得详细更重要**，诚实面对自己。
