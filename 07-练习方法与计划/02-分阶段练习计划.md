# 分阶段练习计划模板

> 知道原则还不够，你需要"今天具体练什么"的可执行计划。本篇按水平阶段给出**练习内容清单 + 每日时间表模板**，可直接套用和调整。

> ⚠️ 这些是参考模板，**请根据自己的时间、目标、进度灵活调整**。有老师指导的，以老师的安排为准。

---

## 一、四个阶段总览

| 阶段 | 大致标志 | 核心任务 |
|------|----------|----------|
| 🌱 **初学（0-1年）** | 认键、识谱、双手简单配合 | 建立姿势、识谱、基本节奏、五指与简单曲子 |
| 🌿 **初级（1-3年）** | 能弹完整简单乐曲、基本音阶 | 音阶琶音体系、车尔尼、巴赫小曲、各调 |
| 🌳 **中级（3-6年）** | 能弹奏鸣曲、复调、多风格 | 全调音阶琶音、创意曲、奏鸣曲、浪漫小品、表现力 |
| 🎓 **高级（6年+）** | 演奏大型作品、有诠释 | 高级技术、大型作品、深入诠释、舞台、个人风格 |

> 时间仅供参考，每个人进度差异极大。**质量比"年限"更重要。**

---

## 二、初学阶段（🌱）

### 学习重点
- 正确的坐姿、手型、放松（[02-01](../02-演奏基础/01-坐姿手型与放松.md)）。
- 认键、音名、识谱基础（[01-01](../01-基础乐理/01-键盘布局与音名.md)、[01-02](../01-基础乐理/02-五线谱与记谱法.md)）。
- 基本节奏与数拍（[01-03](../01-基础乐理/03-节奏与拍子.md)）。
- 手指编号、五指位、简单指法（[02-02](../02-演奏基础/02-手指编号与指法.md)）。
- 连奏/断奏入门（[02-03](../02-演奏基础/03-触键与音色.md)）。

### 教材曲目
- 拜厄 / 小汤普森 / 巴斯蒂安等启蒙教程。
- C、G、F 大调五指练习与简单音阶（单手起）。
- 简单儿歌、民谣、启蒙小曲。

### 每日 20-30 分钟模板
```
5 分钟   放松 + 五指练习 / 简单音阶
10 分钟  启蒙教材当前曲目（分手→合手，慢练）
5-10分钟 识谱/节奏小练习 或 复习已会小曲
2-3分钟  自由弹 / 享受
```

---

## 三、初级阶段（🌿）

### 学习重点
- 音阶琶音体系起步（[03-01](../03-技巧训练/01-音阶与琶音.md)）：常用大小调音阶、主和弦琶音。
- 手指独立与均匀（[03-02](../03-技巧训练/02-手指独立性.md)，温和地练）。
- 踏板入门（[02-04](../02-演奏基础/04-踏板的使用.md)）。
- 视奏习惯养成（[02-05](../02-演奏基础/05-识谱与视奏.md)）。
- 乐理：音程、音阶调式、和弦基础（[01-04](../01-基础乐理/04-音程.md) ~ [01-06](../01-基础乐理/06-和弦基础.md)）。

### 教材曲目
- 车尔尼 599 → 849、布格缪勒 Op.100。
- 哈农（少量、放松地作热身）。
- 巴赫《初级曲集》小步舞曲。
- 克列门蒂/库劳小奏鸣曲。
- 简单的流行/古典小品。

### 每日 30-60 分钟模板
```
5-10分钟  放松 + 音阶琶音（每天轮 2-3 个调）+ 一条哈农
10-15分钟 练习曲（车尔尼/布格缪勒，攻当前难点）
15-25分钟 曲目攻坚（巴赫小曲 / 小奏鸣曲 / 小品，分手分段慢练）
5-10分钟  视奏新材料 + 复习
```

---

## 四、中级阶段（🌳）

### 学习重点
- 全 12 大小调音阶（4 八度、双手、反向）+ 琶音 + 半音阶。
- 复调能力（[04-01](../04-进阶演奏/01-巴洛克与复调.md)）：二部/三部创意曲。
- 奏鸣曲式与古典风格（[04-02](../04-进阶演奏/02-古典时期与奏鸣曲.md)）。
- 浪漫派表现力（[04-03](../04-进阶演奏/03-浪漫派.md)）：歌唱、rubato、踏板。
- 力度句法表现力深化（[03-04](../03-技巧训练/04-力度句法与表现力.md)）。
- 调性和声进行、开始分析作品（[01-07](../01-基础乐理/07-调性与和声进行.md)、[05-01](../05-深入与精通/01-曲式与音乐分析.md)）。
- 背谱方法（[05-03](../05-深入与精通/03-背谱与记忆.md)）。

### 教材曲目
- 车尔尼 299 → 740、莫什科夫斯基 Op.72。
- 巴赫二部/三部创意曲、平均律选曲。
- 莫扎特/海顿/贝多芬奏鸣曲（如《月光》一乐章、《悲怆》）。
- 肖邦圆舞曲/夜曲/前奏曲、舒曼《童年情景》、门德尔松《无词歌》。
- 德彪西易曲、巴托克《小宇宙》。

### 每日 60-90 分钟模板
```
10-15分钟 放松 + 音阶琶音（系统轮转所有调）+ 技术练习
15-20分钟 练习曲（车尔尼等，针对性技术）
30-40分钟 曲目攻坚（复调 + 一首大型/古典 + 一首浪漫，分段慢练攻难点）
10-15分钟 整合音乐性 / 背谱 / 分析
5-10分钟  视奏 + 自由弹
```

---

## 五、高级阶段（🎓）

### 学习重点
- 高级技术（八度、双音、快速跑动、大跳）的精进（[03-05](../03-技巧训练/05-技术难点攻克.md)）。
- 大型作品的驾驭（完整奏鸣曲、叙事曲、协奏曲、组曲）。
- 深入诠释与个人风格（[05-02](../05-深入与精通/02-音乐诠释.md)、[06-03](../06-大师与艺术家/03-个人艺术风格.md)）。
- 舞台表演与心理（[06-02](../06-大师与艺术家/02-舞台表演与心理.md)）。
- 风格的广度（巴洛克到现代、爵士即兴）。
- 高效练习科学的全面运用（[05-04](../05-深入与精通/04-高效练习科学.md)）。

### 教材曲目
- 车尔尼 740、克拉莫、克列门蒂《名手之道》。
- 肖邦/李斯特/德彪西/拉赫等音乐会练习曲（逐步）。
- 完整的贝多芬/肖邦/舒曼大型作品。
- 巴赫平均律、组曲。
- 现代作品、协奏曲、室内乐合作。

### 每日 90-180 分钟模板（可分上下午两段）
```
15-20分钟 放松 + 全调音阶琶音 + 高级技术练习
20-30分钟 高级练习曲 / 技术专项
60-90分钟 曲目攻坚（多部作品，深度打磨难点 + 诠释）
20-30分钟 整合、背谱、模拟演出、分析
10-15分钟 视奏 / 即兴 / 探索新风格
```

## 六、所有阶段通用的每周/每月安排

### 每周
- 5-6 天练习，留 1 天休息（休息也是巩固的一部分）。
- 每周至少 1 次：**完整地"演一遍"**正在打磨的曲子（模拟演出，不停顿）。
- 每周至少 1 次：**录音 + 回放反思**。

### 每月
- 复盘进度：哪些目标达成了？哪些要调整？
- 学一首**全新**的、够得着的"享受型"曲子，保持热情。
- 拓展聆听：认识新的钢琴家/作品（[06-01](../06-大师与艺术家/01-伟大钢琴家.md)）。

---

## 七、动手练习

1. 判断你目前处于哪个阶段，选对应模板。
2. 根据你的**实际可用时间**，把模板按比例缩放成你的每日计划。
3. 写下本周要攻克的 1-2 个具体目标。
4. 安排好"每周模拟演出"和"每周录音"的固定时间。
5. 选一首本月的"享受型新曲"。

---

## 八、小结

- 四阶段：**初学（姿势识谱）→ 初级（音阶练习曲小曲）→ 中级（复调奏鸣曲浪漫+表现力）→ 高级（大型作品+诠释+舞台+风格）**。
- 每阶段给出**学习重点 + 教材曲目 + 每日时间表模板**，按实际时间缩放。
- 通用安排：**每周 5-6 天 + 模拟演出 + 录音反思**；每月**复盘 + 新曲 + 拓展聆听**。
- 时间年限仅参考，**质量与持续性比年限更重要**。
