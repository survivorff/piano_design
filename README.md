# 🎹 钢琴（键盘）完全学习知识库

> 从入门到精通，从精通到大师，从技艺到艺术——一份系统、详尽、可终身使用的钢琴学习与练习宝典。

这个知识库的目标，是陪你走完一条完整的钢琴之路：
**认识乐器 → 理解音乐语言 → 掌握身体技巧 → 驾驭曲目 → 形成个人诠释 → 走向艺术表达。**

无论你是从未碰过琴键的新手，还是想系统补全短板的进阶者，都可以在这里找到对应的阶段。

---

## 📚 如何使用本知识库

1. **按阶段循序渐进**：目录按由浅入深排列，建议新手从 `00` 开始顺序学习。
2. **理论与练习结合**：每学一块理论，立刻去 `07-练习方法与计划` 找对应练习。
3. **不必一次学完**：钢琴是一辈子的事。把它当作一本随时翻阅的手册。
4. **动手最重要**：阅读 1 小时，不如在琴上摸索 10 分钟。

### 🖼️ 关于图示、乐谱与练习工具

- **矢量图示**：关键概念（键盘、五度圈、手型、坐姿、指法、大谱表等）配有 SVG 图示，可在 GitHub 网页直接查看，详见 [assets 说明](./assets/README.md)。
- **可听乐谱示例**：部分练习用 **ABC 文本记谱法**给出，可粘贴到支持 ABC 的在线渲染器中**看谱并听声音**。
- **专业练习**：如何获取正版/免费乐谱（IMSLP、原典版）、使用打谱/练耳/节拍器等软件，详见 [乐谱与练习软件指南](./07-练习方法与计划/08-乐谱与练习软件指南.md)。
- **真实影像示范**：本仓库为开源文本+矢量图，不内嵌版权照片/视频；真人示范、大师课、乐队演出等请参考 [视频、音频与演出资源导航](./07-练习方法与计划/09-视频音频与演出资源.md)（含柏林爱乐数字音乐厅、Musopen、国际比赛存档等可合法访问的资源）。

## 🎯 给不同人的建议起点

- **完全新手**：直接用 [新手 30 天入门计划](./07-练习方法与计划/06-新手30天入门计划.md) 上手，配合 [键盘乐器的历史](./00-入门介绍/01-键盘乐器历史.md) 和 [键盘布局与音名](./01-基础乐理/01-键盘布局与音名.md)。
- **会一点但不系统**：直接看 [01 基础乐理](./01-基础乐理/) 补理论，再到 [03 技巧训练](./03-技巧训练/)。
- **想提升表现力**：重点看 [04 进阶演奏](./04-进阶演奏/) 和 [05 深入与精通](./05-深入与精通/)。

> 💡 **核心理念**：技术是为音乐服务的，音乐是为情感与美服务的。永远不要为了炫技而练琴，而要为了"说出心里想说的话"而练琴。

祝你在琴键上找到属于自己的声音。🎵
