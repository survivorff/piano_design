# 键盘乐器的种类

> 在你买琴或选择练习乐器之前，先搞清楚都有哪些选项，以及它们各自适合谁。

---

## 一、声学钢琴（Acoustic Piano）

真正用琴弦和音板发声的传统钢琴，音色和触感是数码琴难以完全替代的。

### 1. 三角钢琴（Grand Piano）
- 琴弦水平放置，击弦机靠重力自然回落，**回弹快、控制精准**。
- 按尺寸分：
  - **音乐会三角琴（Concert Grand）**：约 2.7 米，用于大音乐厅。
  - **演奏级三角琴**：约 2.1–2.3 米。
  - **室内三角琴 / 小三角（Baby Grand）**：约 1.4–1.7 米。
- 优点：音色最丰富、动态范围最大、触键最灵敏。
- 缺点：贵、占地大。

### 2. 立式钢琴（Upright / Vertical Piano）
- 琴弦垂直放置，节省空间，是家庭最常见的选择。
- 击弦机靠**弹簧**辅助回落（因为重力方向不对），回弹略逊于三角琴。
- 按高度分：高琴（>120cm，音色更好）、中型、矮琴（音色相对受限）。
- 优点：占地小、价格友好、音色仍然真实。
- 缺点：动态和触感不如三角琴。

## 二、数码 / 电子键盘乐器

### 1. 数码钢琴（Digital Piano）
- 用**录制的真实钢琴采样**或**物理建模**发声。
- 好的型号配有 **88 键 + 配重键（weighted / hammer action）**，模拟真实钢琴的手感。
- 优点：
  - 可调音量、可戴耳机（**不扰邻**）。
  - 永远不跑音、几乎免维护。
  - 内置多种音色、节拍器、录音功能。
  - 比声学琴便宜、轻便。
- 缺点：手感和音色细腻度仍略逊于优质声学琴（高端型号差距已很小）。
- **对新手和公寓住户，这是性价比极高的选择。**

### 2. 舞台钢琴（Stage Piano）
- 专为现场演出设计，音色丰富、接口齐全，便于携带和连接音响。

### 3. 合成器（Synthesizer）
- 通过电子方式**生成和塑造声音**，能创造钢琴之外的无限音色。
- 多用于流行、电子、影视配乐。
- 键盘通常较轻（非配重），更适合快速演奏电子音色，不适合练古典触键。

### 4. MIDI 键盘控制器（MIDI Controller）
- **本身不发声**，只发送演奏信号给电脑，由音乐软件（DAW）产生声音。
- 是电脑音乐制作的核心工具。

### 5. 电钢琴（Electric Piano，如 Rhodes、Wurlitzer）
- 一种特定的复古乐器，用机械装置 + 电磁拾音，音色温暖独特。
- 是爵士、灵魂乐、放克的经典音色。

## 四、我该选哪一种？（决策指南）

| 你的情况 | 推荐 |
|----------|------|
| 完全新手 / 预算有限 / 住公寓怕吵 | **88 键配重数码钢琴** |
| 认真长期学古典、有空间和预算 | **立式或三角声学钢琴** |
| 想做流行/电子音乐制作 | **MIDI 键盘 + 电脑软件** 或 **合成器** |
| 经常演出、需要便携 | **舞台钢琴** |
| 孩子启蒙、不确定能否坚持 | **数码钢琴**（投入小、易转手） |

> 🔑 **给新手最重要的一条建议**：无论数码还是声学，**一定要选 88 个全尺寸配重琴键**。轻飘飘的非配重小键盘会养成错误的手指力量习惯，将来转到真钢琴会很吃力。

---

## 五、小结

- 声学钢琴分**三角**（音色最佳）和**立式**（实用省地）。
- 数码钢琴是新手和公寓族的理想入门选择，关键看是否 **88 键 + 配重**。
- 合成器、MIDI 键盘服务于电子音乐与制作。
- 选琴的核心：**配重全尺寸键盘 + 你的预算和空间 + 你的音乐目标**。
