# 键盘布局与音名

> 这是你弹琴的第一张地图。一旦理解了键盘的规律，88 个键就不再吓人，而是井然有序。

---

## 一、黑键的规律：你的定位锚点

看一眼键盘，你会发现黑键不是均匀排列的，而是**两个一组**和**三个一组**交替出现：

```
 ┌─┐ ┌─┐  ┌─┐ ┌─┐ ┌─┐    ┌─┐ ┌─┐  ┌─┐ ┌─┐ ┌─┐
 │ │ │ │  │ │ │ │ │ │    │ │ │ │  │ │ │ │ │ │
 │ │ │ │  │ │ │ │ │ │    │ │ │ │  │ │ │ │ │ │
─┘ └─┘ └──┘ └─┘ └─┘ └────┘ └─┘ └──┘ └─┘ └─┘ └──
 C  D  E  F  G  A  B  C ...
   2黑键      3黑键
```

- **两个黑键一组**：左边那个白键就是 **C（do）**。
- 利用这个规律，你能瞬间在任何位置找到 C，进而找到所有音。

> 🔑 **第一个要记住的事**：紧挨着"两个黑键"左边的白键 = **C（中文：do）**。这是整个键盘的基准点。

---

## 二、白键的音名

白键从 C 开始，按字母循环：

**C – D – E – F – G – A – B – （回到）C ...**

对应唱名（首调/固定调常用）：

| 字母音名 | C | D | E | F | G | A | B |
|---------|---|---|---|---|---|---|---|
| 唱名 | do | re | mi | fa | sol | la | si(ti) |

这 7 个音不断循环，覆盖整个键盘。

---

## 三、八度（Octave）

- 从一个 C 到下一个 C，叫做一个**八度**（因为跨越了 8 个白键音）。
- 高八度的音听起来是"同一个音的更高版本"——因为它的振动频率正好是**两倍**。
- 标准 88 键钢琴覆盖 **7 又 1/4 个八度**。

### 音区命名（科学音高记号）
为了区分不同八度的同名音，用数字标记：
- **中央 C = C4**（钢琴正中间、品牌标志附近那个 C）。
- 往右每过一个 C，数字 +1（C5、C6...）；往左 −1（C3、C2...）。
- 钢琴最低音是 **A0**，最高音是 **C8**。

> 🔑 **中央 C（C4）** 是识谱的核心参照点，记住它的位置。

---

## 四、黑键的音名：升号与降号

黑键没有自己独立的字母名，它们是相邻白键的"升高"或"降低"版本：

- **升号 ♯（sharp）**：升高半音（往右一个键）。
- **降号 ♭（flat）**：降低半音（往左一个键）。

所以**同一个黑键有两个名字**（这叫"等音异名"，enharmonic）：

| 黑键位置 | 名字一 | 名字二 |
|----------|--------|--------|
| C 和 D 之间 | C♯（升C） | D♭（降D） |
| D 和 E 之间 | D♯（升D） | E♭（降E） |
| F 和 G 之间 | F♯（升F） | G♭（降G） |
| G 和 A 之间 | G♯（升G） | A♭（降A） |
| A 和 B 之间 | A♯（升A） | B♭（降B） |

叫哪个名字，取决于乐曲所在的**调**（后面音阶调式章节会讲）。

---

## 五、半音与全音（音乐的"基本距离单位"）

这是**整个乐理的基石**，务必理解透：

- **半音（Semitone / Half step）**：键盘上**相邻的两个键**（不分黑白）之间的距离。这是最小的距离单位。
  - 例：C → C♯，E → F（注意！E 和 F 之间没有黑键，它们直接相邻，所以是半音）。
- **全音（Whole tone / Whole step）= 两个半音**：中间隔一个键。
  - 例：C → D（中间隔着 C♯），F → G。

### 两个"陷阱"要特别注意
键盘上有两处白键之间**没有黑键**，它们本身就是半音关系：
- **E → F**（半音）
- **B → C**（半音）

> 🔑 记住：**E-F 和 B-C 之间是半音**。这是初学者最容易混淆的地方，也是理解音阶构成的关键。

---

## 六、动手练习

1. **找 C**：闭上眼，随便从一个"两黑键"组左侧按下，确认是不是 C，反复练到秒找。
2. **唱名定位**：在键盘任意一个八度内，依次按 C-D-E-F-G-A-B 并念出唱名。
3. **找中央 C**：每次坐下先找到 C4，建立"原点"意识。
4. **半音/全音辨认**：随便指两个相邻键，判断是半音还是全音；再练全音。
5. **黑键命名**：指一个黑键，说出它的两个名字（如"升C / 降D"）。

---

## 七、小结

- 黑键 **2 个 + 3 个** 交替，**两黑键左侧 = C**，这是定位锚点。
- 白键循环：**C D E F G A B**。
- **八度** = 同名音的高低版本；**中央 C = C4** 是参照原点。
- 黑键 = 相邻白键的 **♯（升）/ ♭（降）**，一键两名。
- **半音** = 相邻键；**全音** = 隔一键；牢记 **E-F、B-C 是半音**。
