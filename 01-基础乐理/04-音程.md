# 音程（Intervals）

> 音程是两个音之间的"距离"。它是和弦、音阶、和声的building block。理解了音程，你就能"听懂"音乐里两个音为什么放在一起好听或紧张。

---

## 一、什么是音程

**音程 = 两个音之间的音高距离。**

音程有两个属性，必须**两个一起说**才完整：
1. **度数（Number）**：跨越了几个字母音名。
2. **性质（Quality）**：大、小、纯、增、减。

例如"大三度""纯五度"——前面是性质，后面是度数。

---

## 二、度数：先数字母

度数 = 从低音到高音，**包含首尾**地数字母音名的个数。

| 从 C 到 | 跨越字母 | 度数 |
|---------|----------|------|
| C → C | C（同一个音） | 一度（unison，纯一度） |
| C → D | C-D | 二度 |
| C → E | C-D-E | 三度 |
| C → F | C-D-E-F | 四度 |
| C → G | C-D-E-F-G | 五度 |
| C → A | …A | 六度 |
| C → B | …B | 七度 |
| C → 高C | …C | 八度（octave） |

> 🔑 **数度数只看字母，不管升降号**。C 到 E 是三度，C 到 E♭ 也是三度（只是性质不同）。

---

## 三、性质：再数半音

光知道度数不够，因为同样是"三度"，听起来可能明亮（大三度）或忧郁（小三度）。性质由**半音数**精确决定。

### 五种性质
- **纯（Perfect, P）**：只用于 一度、四度、五度、八度。
- **大（Major, M）** 和 **小（minor, m）**：用于 二度、三度、六度、七度。
- **增（Augmented, A）**：比纯/大再宽**半音**。
- **减（Diminished, d）**：比纯/小再窄**半音**。

### 常用音程的半音数对照表

| 音程 | 半音数 | 从 C 起的例子 | 听感 |
|------|--------|---------------|------|
| 纯一度 | 0 | C–C | 完全相同 |
| 小二度 | 1 | C–D♭ | 极不协和、紧张（电影惊悚配乐） |
| 大二度 | 2 | C–D | 略紧张 |
| 小三度 | 3 | C–E♭ | 忧郁、柔和（小调色彩） |
| 大三度 | 4 | C–E | 明亮、愉快（大调色彩） |
| 纯四度 | 5 | C–F | 稳定、空旷 |
| 增四度/减五度（三全音） | 6 | C–F♯ | 极不稳定，"魔鬼音程" |
| 纯五度 | 7 | C–G | 非常稳定、空旷（摇滚强力和弦） |
| 小六度 | 8 | C–A♭ | 略忧郁 |
| 大六度 | 9 | C–A | 开阔、甜美 |
| 小七度 | 10 | C–B♭ | 紧张待解决（属七和弦） |
| 大七度 | 11 | C–B | 尖锐、现代、爵士感 |
| 纯八度 | 12 | C–高C | 同音的高低版本 |

> 🔑 **三全音（Tritone，6个半音）**：中世纪被称为"音乐中的魔鬼"，因为它极度不稳定、刺耳。但它正是大调音阶里 fa-si 的关系，是推动和声"想要解决"的关键张力来源。

---

## 四、协和音程 vs 不协和音程

- **协和（Consonant）**：听起来稳定、悦耳、放松。
  - 纯一度、纯八度、纯五度（最协和）；大小三度、大小六度（柔和协和）。
- **不协和（Dissonant）**：听起来紧张、不稳定、"想要往别处走"。
  - 二度、七度、三全音。

> 💡 **音乐的美感来自张力与放松的交替**。不协和不是"坏"，而是制造期待和情感的工具——它让随后的"解决（resolution）"显得动人。

---

## 五、旋律音程 vs 和声音程

- **旋律音程（Melodic）**：两个音**先后**弹出（如旋律线条中的跳进）。
- **和声音程（Harmonic）**：两个音**同时**弹出（构成和弦的基础）。

---

## 六、转位（Inversion）

把音程的低音翻高八度（或高音降八度），就得到它的**转位**。规律很美：
- 度数转位：两数相加 = **9**。（三度↔六度，二度↔七度，四度↔五度）
- 性质转位：**大↔小，增↔减，纯↔纯**。
- 例：大三度（C–E）转位 → 小六度（E–高C）。

---

## 七、为什么音程如此重要？

1. **构建和弦**：大三和弦 = 大三度 + 小三度叠加。理解音程，和弦就不用死记。
2. **听辨能力（视唱练耳）**：能听出音程，你就能扒谱、即兴、辨别走调。
3. **理解和声张力**：知道哪些音程紧张、哪些放松，才能理解作曲家如何营造情绪。
4. **移调与即兴**：以音程关系思考，能让你脱离"固定音名"自由演奏。

### 用熟悉的歌记音程（练耳小窍门）
- 小二度↓：电影《大白鲨》主题。
- 大二度↑：《小星星》开头"一闪"。
- 大三度↑：《欢乐颂》开头。
- 纯四度↑：《婚礼进行曲》开头、《茉莉花》。
- 纯五度↑：《小星星》"一闪一闪"到"亮晶晶"、《星球大战》主题。
- 纯八度↑：《Somewhere Over the Rainbow》"Some-where"。

---

## 八、动手练习

1. **构造音程**：从 C 开始，弹出每一种音程并听辨色彩。
2. **大小三度对比**：交替弹 C-E（大）和 C-E♭（小），感受"明—暗"。
3. **唱音程**：先弹再唱，建立内在听觉。
4. **找张力**：弹三全音（C-F♯），再把它"解决"到 B-C/E，感受放松。
5. **用歌记忆**：把上面的"歌曲对照"练熟，听到音程能秒认。

---

## 九、小结

- 音程 = **度数（数字母）+ 性质（数半音）**，两者缺一不可。
- 性质有 **纯 / 大 / 小 / 增 / 减** 五种。
- **协和**给人稳定，**不协和**制造张力——音乐靠两者交替产生美。
- 音程是**和弦、音阶、听辨、即兴**的共同基础。
