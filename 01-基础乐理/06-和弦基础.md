# 和弦基础（Chords）

> 和弦是"同时响起的几个音"，是音乐的"色彩与厚度"。学会和弦，你就能给旋律配伴奏、看懂和弦谱、即兴弹唱。

---

## 一、什么是和弦

**和弦 = 三个或更多音同时发声。**

最基础、最重要的和弦类型是**三和弦（triad）**——由三个音按"三度叠加"构成。

---

## 二、三和弦的构造：三度叠加

从一个音（**根音 root**）开始，往上叠两个三度：

```
        5音（五音）
       ↑ 三度
        3音（三音）
       ↑ 三度
        1音（根音 root）
```

三个音分别叫：
- **根音（Root）**：和弦的名字来源、最低的基础音。
- **三音（Third）**：决定和弦"明/暗"（大/小）。
- **五音（Fifth）**：增加稳定和厚度。

> 🔑 **三音是和弦的"情绪开关"**：大三度→大和弦（明亮），小三度→小和弦（忧郁）。

---

## 三、四种基本三和弦

用从根音算起的音程来区分：

| 类型 | 结构（从根音） | 内部叠加 | 色彩 | C 上的例子 |
|------|----------------|----------|------|------------|
| **大三和弦 Major** | 大三度 + 纯五度 | 大3 + 小3 | 明亮、稳定、快乐 | C-E-G |
| **小三和弦 minor** | 小三度 + 纯五度 | 小3 + 大3 | 忧郁、柔和 | C-E♭-G |
| **增三和弦 Augmented** | 大三度 + 增五度 | 大3 + 大3 | 紧张、悬疑、不稳定 | C-E-G♯ |
| **减三和弦 diminished** | 小三度 + 减五度 | 小3 + 小3 | 紧张、阴暗、不安 | C-E♭-G♭ |

**记忆要点**：
- 大、小是常用主力。
- 增、减用于制造张力和过渡，较少作为"落脚点"。

---

## 四、和弦的转位（Inversions）

同样三个音，可以换不同的音在最低，得到不同"排列"。以 C 大三和弦（C-E-G）为例：

| 名称 | 最低音 | 排列 | 标记 |
|------|--------|------|------|
| **原位（Root position）** | 根音 C | C-E-G | |
| **第一转位（1st inversion）** | 三音 E | E-G-C | C/E |
| **第二转位（2nd inversion）** | 五音 G | G-C-E | C/G |

**为什么要转位？**
- 让和弦之间的连接更**平滑**（手不用大跳）。
- 改变声音的"重心"和色彩。
- 制造低音线条（bass line）。

> 💡 标记 "C/E" 读作"C over E"，意思是 C 和弦、但最低音是 E。流行和弦谱里很常见。

---

## 五、七和弦（Seventh Chords）—— 更丰富的色彩

在三和弦上**再叠一个三度**（即根音上方的七度音），得到四个音的七和弦。它们更"丰满、有味道"，是爵士和流行的核心。

| 类型 | 结构 | 色彩/用途 | C 上的例子 |
|------|------|-----------|------------|
| **大七和弦 (Maj7)** | 大三和弦 + 大七度 | 柔和、梦幻、爵士、城市感 | C-E-G-B |
| **属七和弦 (7 / Dominant 7)** | 大三和弦 + 小七度 | 强烈的"想解决"张力，蓝调 | C-E-G-B♭ |
| **小七和弦 (m7)** | 小三和弦 + 小七度 | 柔和忧郁、放松 | C-E♭-G-B♭ |
| **小大七 (mMaj7)** | 小三和弦 + 大七度 | 紧张神秘（悬疑配乐） | C-E♭-G-B |
| **半减七 (m7♭5 / ø7)** | 减三和弦 + 小七度 | 不安、过渡 | C-E♭-G♭-B♭ |
| **减七 (dim7)** | 减三和弦 + 减七度 | 极度紧张、戏剧性转折 | C-E♭-G♭-B♭♭(A) |

> 🔑 **属七和弦（如 G7）** 是调性音乐的"引擎"，它强烈地推动音乐回到主和弦（详见下一篇 [调性与和声进行](./07-调性与和声进行.md)）。

---

## 六、和弦的命名与记号

流行 / 爵士常用字母记号（以 C 为根音）：

| 记号 | 含义 |
|------|------|
| C | C 大三和弦 |
| Cm / Cmin / C− | C 小三和弦 |
| Caug / C+ | C 增三和弦 |
| Cdim / C° | C 减三和弦 |
| C7 | C 属七 |
| Cmaj7 / CΔ | C 大七 |
| Cm7 | C 小七 |
| Csus4 | 挂留和弦（用纯四度替代三音，C-F-G，悬而未决） |
| Csus2 | 用大二度替代三音（C-D-G） |
| C6 | 大三和弦加六度（C-E-G-A，明亮甜美） |
| C9, C11, C13 | 加入更高叠置音的扩展和弦（爵士常用） |
| Cadd9 | 在三和弦上直接加九度（C-E-G-D） |

---

## 七、和弦的"扩展"与"挂留"（简介）

- **挂留和弦（sus）**：把三音换成二度或四度，产生"悬而未决、期待解决"的开放感。常解决回普通和弦。
- **加音和弦（add）**：在三和弦基础上加一个色彩音（如 add9），不改变基本性质但更丰富。
- **扩展和弦（9/11/13）**：继续往上叠三度，色彩越来越复杂、越来越"爵士"。

这些进阶用法在 [爵士、流行与即兴](../04-进阶演奏/05-爵士流行与即兴.md) 会深入。

---

## 八、如何用和弦伴奏（实用入门）

1. **找出歌曲的调**（看调号 / 听主音）。
2. **用左手弹和弦根音或整个和弦**，右手弹旋律。
3. 最常用的伴奏型：
   - **柱式和弦**：左手同时按下整个和弦（简单稳定）。
   - **分解和弦（琶音式）**：把和弦音一个个弹出（如 C-G-E-G），更流动优美。
   - **阿尔贝蒂低音**：低-高-中-高循环（古典风，莫扎特常用）。
4. 先从只用 **3-4 个和弦**的简单歌曲开始（很多流行歌只用 C-G-Am-F）。

---

## 九、动手练习

1. **造四种三和弦**：在 C 上弹出大、小、增、减，听辨色彩差异。
2. **大小转换**：弹 C 大（C-E-G），把 E 降半音变 C 小（C-E♭-G），反复感受。
3. **练转位**：把 C 和弦弹出原位、一转、二转，连起来。
4. **七和弦听感**：弹 Cmaj7、C7、Cm7，体会"梦幻—张力—忧郁"。
5. **四和弦伴奏**：用 C–G–Am–F 循环，左手和弦右手随便哼旋律。

---

## 十、小结

- 和弦 = 多音同响；最基础是**三和弦**（根-三-五，三度叠加）。
- 四种三和弦：**大（明）、小（暗）、增/减（张力）**，三音定明暗。
- **转位**让和弦连接更平滑、低音更有线条。
- **七和弦**带来丰富色彩，**属七**是推动解决的引擎。
- 掌握字母记号和基本伴奏型，就能开始弹唱与配伴奏。
