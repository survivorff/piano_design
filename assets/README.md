# assets — 图示与素材说明

本文件夹存放知识库中使用的**矢量图示（SVG）**。

## 为什么用 SVG？

- **GitHub 原生渲染**：在网页和大多数 Markdown 阅读器里能直接显示。
- **矢量、清晰**：任意缩放不模糊。
- **可版本控制**：本质是文本，便于追踪修改、协作。
- **轻量**：体积小，加载快。

## 关于真实照片与视频

本知识库是纯文本/矢量图的开源仓库，**不内嵌受版权保护的照片或视频**。
对于需要真实影像示范的内容（如手型实拍、大师演奏、乐器内部结构照片），
请参阅 [学习资源与工具](../07-练习方法与计划/04-学习资源与工具.md) 与
[乐谱与练习软件指南](../07-练习方法与计划/08-乐谱与练习软件指南.md)，
其中给出了权威外部资源的获取途径与说明。

## 想新增/修改图示？

SVG 是纯文本，可用任意文本编辑器或矢量绘图软件（如 Inkscape）打开编辑。
修改后在 Markdown 中用 `![描述](../assets/图示文件名)` 的形式引用即可。
