## 二、速度术语（Tempo）

| 术语 | 含义（大致 BPM） |
|------|------------------|
| Largo | 广板，很慢（40-60） |
| Lento | 慢板 |
| Adagio | 柔板，慢而从容（66-76） |
| Andante | 行板，行走的速度（76-108） |
| Moderato | 中板（108-120） |
| Allegretto | 稍快板 |
| Allegro | 快板（120-168） |
| Vivace | 活泼地、很快 |
| Presto | 急板，很快（168-200） |
| Prestissimo | 最急板，极快 |

### 速度变化
| 术语 | 含义 |
|------|------|
| accelerando（accel.） | 渐快 |
| ritardando（rit.）/ rallentando（rall.） | 渐慢 |
| ritenuto（riten.） | 突然放慢 |
| a tempo | 回到原速 |
| rubato | 自由伸缩的速度 |
| fermata（𝄐） | 延长记号，自由延长该音/休止 |

## 三、基础乐理术语

| 术语 | 含义 |
|------|------|
| 八度（Octave） | 同名音的高低版本，频率2倍 |
| 半音（Semitone） | 相邻两键的距离，最小单位 |
| 全音（Whole tone） | 两个半音 |
| 音程（Interval） | 两音之间的距离（度数+性质） |
| 音阶（Scale） | 按音高排列的一组音 |
| 大调（Major） | 明亮的音阶（全全半全全全半） |
| 小调（Minor） | 忧郁的音阶（自然/和声/旋律三种） |
| 调式（Mode） | 教会调式等不同音阶色彩 |
| 五声音阶（Pentatonic） | 5个音的音阶 |
| 半音阶（Chromatic） | 全部相邻半音 |
| 升号（Sharp ♯） | 升高半音 |
| 降号（Flat ♭） | 降低半音 |
| 还原号（Natural ♮） | 取消升降 |
| 调号（Key signature） | 谱号后标明全曲升降的记号 |
| 五度圈（Circle of fifths） | 调性关系的环形图 |
| 关系大小调 | 共用调号的大调与小调 |

## 四、和弦与和声术语

| 术语 | 含义 |
|------|------|
| 和弦（Chord） | 多个音同时发声 |
| 三和弦（Triad） | 根-三-五三度叠加的三音和弦 |
| 七和弦（Seventh chord） | 三和弦+七度的四音和弦 |
| 根音（Root） | 和弦的基础音 |
| 转位（Inversion） | 非根音作最低音的排列 |
| 大三和弦（Major triad） | 明亮（大三度+小三度） |
| 小三和弦（minor triad） | 忧郁（小三度+大三度） |
| 增/减三和弦 | 紧张、不稳定 |
| 属七和弦（Dominant 7） | 强烈想解决的七和弦 |
| 大七/小七和弦 | 爵士常用色彩和弦 |
| 挂留和弦（sus） | 用2/4度替代三音 |
| 终止式（Cadence） | 乐句结尾的和声公式 |
| 罗马数字分析 | 用 I-ii-iii… 标记和弦功能 |
| 主/下属/属（T/S/D） | 三大和声功能 |
| 离调/转调（Modulation） | 临时强调/转移到别的调 |
| voicing | 和弦音的排列方式（爵士常用） |

## 五、记谱术语

| 术语 | 含义 |
|------|------|
| 五线谱（Staff） | 5线4间的记谱系统 |
| 大谱表（Grand staff） | 钢琴用的高音+低音双谱表 |
| 高音谱号（Treble clef 𝄞） | G谱号，右手 |
| 低音谱号（Bass clef 𝄢） | F谱号，左手 |
| 加线（Ledger line） | 超出五线的短线 |
| 中央C（Middle C / C4） | 大谱表中央的参照音 |
| 小节（Measure / Bar） | 小节线分割的单位 |
| 拍号（Time signature） | 每小节拍数与单位拍 |
| 音符（Note） | 表示音高与时值 |
| 休止符（Rest） | 表示沉默时长 |
| 附点（Dot） | 时值增加一半 |
| 连音线/延音线（Tie） | 连接同音高，时值相加 |
| 圆滑线（Slur） | 乐句/连奏标记 |
| 反复记号（Repeat） | 重复演奏的段落 |

## 六、演奏技术术语

| 术语 | 含义 |
|------|------|
| 指法（Fingering） | 用哪根手指弹（1-5编号） |
| 穿指（Thumb under） | 上行时拇指从下穿过 |
| 跨指（Cross over） | 下行时手指越过拇指 |
| 五指位（Five-finger position） | 5指放相邻5音的手位 |
| 触键（Touch） | 手指接触琴键的方式 |
| 连奏（Legato） | 音音相连，靠手指接力 |
| 断奏（Staccato） | 短促、有弹性 |
| 非连奏（Non-legato） | 清楚分开但不短促 |
| 声部平衡（Voicing/Balance） | 让旋律浮出、伴奏退后 |
| 句法（Phrasing） | 乐句的呼吸与塑形 |
| rubato | 自由伸缩速度 |
| agogic | 用时值延长来强调 |
| 装饰音（Ornament） | 颤音、波音、回音、倚音等 |
| 颤音（Trill） | 两个相邻音快速交替 |
| 波音（Mordent） | 与上/下邻音快速一次交替 |
| 回音（Turn） | 围绕主音的装饰音型 |
| 八度（Octave technique） | 用1-5指弹八度 |
| 音块（Tone cluster） | 一片相邻音同时奏响 |

## 七、踏板术语

| 术语 | 含义 |
|------|------|
| 延音踏板（Damper/Sustain） | 右踏板，抬起所有制音器 |
| 弱音踏板（Una corda/Soft） | 左踏板，柔和音色 |
| 持音踏板（Sostenuto） | 中踏板，选择性延音 |
| 切分踏板（Syncopated pedal） | 手下-脚抬-脚落的换踏板法 |
| 半踏板（Half pedal） | 部分踩下，部分消音 |
| 颤踏板（Flutter pedal） | 快速抖动踏板 |
| Ped. / ✲ | 踩下 / 松开踏板记号 |

| 创意曲（Invention） | 巴赫的二/三声部教学小曲 |
| 前奏曲（Prelude） | 引子性/自由的小品 |

## 十、风格与时期术语

| 术语 | 含义 |
|------|------|
| 巴洛克（Baroque, 1600-1750） | 复调、装饰、巴赫亨德尔 |
| 古典（Classical, 1750-1820） | 清晰均衡、海顿莫扎特贝多芬 |
| 浪漫（Romantic, 1820-1900） | 情感歌唱、肖邦李斯特 |
| 印象派（Impressionism） | 色彩氛围、德彪西拉威尔 |
| 现代/近现代（Modern） | 20世纪多元探索 |
| 新古典主义（Neoclassicism） | 现代手法回归古典结构 |
| 十二音/序列（Serialism） | 无调性的组织技法 |
| 极简主义（Minimalism） | 重复音型渐变 |
| 爵士（Jazz） | 即兴、swing、丰富和声 |
| 蓝调（Blues） | 12小节进行+蓝调音阶 |
| swing | 摇摆的不平均八分节奏 |
| ii-V-I | 爵士核心和声进行 |
| 和弦谱（Lead sheet） | 旋律+和弦记号的谱 |
| 即兴（Improvisation） | 当场创作演奏 |
| 本真演奏（HIP） | 用历史乐器与惯例演奏 |
| 阿尔贝蒂低音（Alberti bass） | 古典常用分解和弦伴奏型 |
